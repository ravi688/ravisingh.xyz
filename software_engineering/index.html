Modifying/Extending other's working code

If the existing other's code working for all the inputs and your modifications/extensions aren't working, that probably indicates that you didn't understand the earlier code well. It might be as frustrating that you start consider the existing code buggy, but if the earlier code works fine for all the inputs then it is not buggy, keep this in mind. And try to understand the code well by asking yourself contrived questions and if you get answers which are satisfactory to you then you'll definitely get a aha moment!
